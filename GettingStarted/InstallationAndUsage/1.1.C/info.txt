	  Example C: Installing using the Vue Command Line Interface (vue-cli)

Description: We learn how to install different Vue.js templates and how which one is used. In this folder, I have 2 examples of the most used Vue.js templates.

Steps: We create a basic work flow by using the vue-cli. There are many templates to choose from.
In the console:
1. npm install -g vue-cli
2. vue init <template-name> <project-name>
       Ex: vue init webpack vuejsfun

We can also get download templates in two ways:
1. Fork a template on github and in the console:
						vue init username/repo my-project
2. Clone the template from github and in the console:
						    vue init ~/fs/path/to-custom-template my project


References: Vue.js Official Documentation on Installation - https://vuejs.org/v2/guide/installation.html
	    NPM Vue-CLI - https://www.npmjs.com/package/vue-cli
